# **App Name**: Zenith Fitness Hub

## Core Features:

- User Authentication & Profiles: Secure user registration, login, and personalized profile management, allowing users to view and update their personal information.
- Workout Scheduling & Booking: Users can browse available workout sessions, view details, book slots, and manage their scheduled bookings directly from the application.
- Product Catalog & Basic Cart: A browsable catalog of club-related products (e.g., apparel, supplements), allowing users to add items to a temporary shopping cart.
- Club News Feed: Display a dynamic news feed with updates, announcements, and photos published by the club administration.
- Admin Dashboard (MVP): A simplified administrator interface for managing user accounts and workout schedules, along with basic operational overview.
- AI-Powered Workout Recommendations: A tool that uses AI to suggest personalized workout routines and training plans to users based on their recorded activities and goals.
- Manual Attendance Check-in: Allows club staff to manually record member attendance for workout sessions, updating their check-in status.

## Style Guidelines:

- The overall theme is dark, creating a sophisticated and immersive experience.
- Primary color: A vibrant, rich purple (#B37DE8) to symbolize ambition and a premium, modern feel. This color contrasts effectively against the dark background.
- Background color: A very dark, muted purplish-gray (#26212C) derived from the primary hue, ensuring readability and a consistent dark aesthetic.
- Accent color: A deep, expressive blue (#3333CC) used for calls to action, interactive elements, and key highlights, providing a distinct contrast and complementing the primary purple.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and impactful appearance that resonates with the glassmorphism aesthetic.
- Body text font: 'Inter' (sans-serif) for its clean, objective readability, ensuring a seamless user experience for longer text blocks and detailed information.
- Employ a set of sleek, modern, and minimalistic line-based icons to align with the overall sophisticated, clean aesthetic and glassmorphism elements.
- Utilize a mobile-first, responsive design with a persistent sidebar for primary navigation and a clean top navbar for essential actions. Dashboard cards should clearly present information with glassmorphism effects for depth.
- Incorporate subtle, smooth transition animations for state changes and route navigations. Provide clear loading states for dynamic content and use toast notifications for user feedback and important alerts.
