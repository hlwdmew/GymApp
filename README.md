качалка.com

Премиальная система управления фитнес-клубом на базе Next.js 15, GenAI и Firebase.

## 🚀 Как запустить проект локально

Если вы скачали проект на компьютер:

1. **Установите зависимости**:
   ```bash
   npm install
   ```

2. **Настройте переменные окружения**:
   Создайте файл `.env.local` в корне проекта и добавьте туда ваши ключи из `src/firebase/config.ts` (хотя они уже прописаны в коде для удобства).

3. **Запустите сервер разработки**:
   ```bash
   npm run dev
   ```
   Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 🔑 Как зайти в Админ-панель
В системе реализована автоматическая проверка роли при регистрации:
1. Зайдите в раздел **Регистрация**.
2. Введите Email, содержащий слово `admin` (например: `admin@kachalka.com`).
3. Введите любой пароль.
4. После входа вам будут доступны разделы "Аналитика", "Управление залом" и "Управление лентой".

**Альтернативный способ:**
Вы можете сменить роль в настройках своего **Профиля** после авторизации (я добавил туда удобный переключатель).

## 🛠 Технологии
- **Frontend**: Next.js 15 (App Router), Tailwind CSS, ShadCN UI.
- **Backend**: Firebase Auth & Firestore.
- **AI**: Google Gemini через Genkit (для планов питания и тренировок).

## 📦 Экспорт на GitHub
1. Скачайте проект как ZIP через кнопку в Firebase Studio.
2. Распакуйте и инициализируйте git: `git init`.
3. Создайте репозиторий на GitHub и следуйте инструкциям по `git push`.
